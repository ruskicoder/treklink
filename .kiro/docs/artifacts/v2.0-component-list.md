# TrekLink v2.0 — Complete Component List (BOM)

> **Revision:** 2.1  
> **Date:** 2026-06-17  
> **Assembly:** PCBA via JLCPCB (SMD components on PCB)  
> **Hand-Solder:** OLED display (direct to pads), external button breakout wires, external USB-C breakout wires, buzzer, vibrator, battery leads, antenna pigtail  
> **Design Tool:** KiCad  

---

## Confirmed Design Decisions (Rev 2.1 - Freelancer Actual)

| # | Decision | Details / Impact |
|---|----------|------------------|
| D1 | PCB Layout | **Half-size PCB (52.5 × 70.0 mm)**. Left side is battery compartment. |
| D2 | GPS | NEO-M9N (premium 4-constellation GNSS). |
| D3 | PCBA Scope | All surface-mount ICs, passives, and castellated modules are on the PCBA. OLED, main buttons, and USB-C port are off-board/breakout to minimize stress. |
| D4 | Design Tool | **KiCad (v10.0 format)**. |
| D5 | Antenna | IPEX to panel-mount SMA pigtail for LoRa. |
| D6 | Interop | Meshtastic protocol on 433 MHz band. |
| D7 | Power Button | Soft-latch circuit gating the VBAT_OUT power rail. |
| D8 | OLED Mount | 0.96" SSD1306 OLED module direct-soldered to 4-pin plated through-holes on PCB. |
| D9 | Buttons | 6× small SMD tactile test buttons on-board + breakout header J4 for hand-wired external buttons. |
| D10 | USB-C | Off-board breakout wired to J1 breakout pads. CC1/CC2 pull-downs on PCB. |
| D11 | GPS Antenna | Passive ceramic patch antenna internally mounted, connected via U.FL. |
| D12 | Active Antenna | Bias-T circuit populated via 0R jumper R11 (default DNP). |
