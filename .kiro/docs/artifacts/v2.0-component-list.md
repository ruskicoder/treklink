# TrekLink v2.0 — Complete Component List (BOM)

> **Revision:** 2.0  
> **Date:** 2026-05-28  
> **Batch Size:** 7 units  
> **Assembly:** PCBA via JLCPCB (SMD components on PCB)  
> **Hand-Solder:** OLED display (direct to pads), 5× external button breakout wires, 4× external USB-C breakout wires, buzzer, vibrator, battery leads, antenna pigtail, GPS CR1220 battery  
> **Design Tool:** KiCad (primary)  
> **Decisions Reference:** `answers.md` & `enclosure-specifications.md`

## Confirmed Design Decisions (Rev 2.0)

| # | Decision | Details / Impact |
|---|----------|------------------|
| D1 | PCB Layout | **Half-size PCB (51.8 × 69.1 mm)** shifted right. Left side is a **44.0 mm wide battery zone** for 1S2P 21700. |
| D2 | GPS | NEO-M9N (premium 4-constellation GNSS). |
| D3 | PCBA Scope | All surface-mount ICs, passives, and castellated modules are on the PCBA. OLED, main buttons, and USB-C port are off-board/breakout to minimize enclosure fitting stress. |
| D4 | Design Tool | **KiCad primary (v8.0+)**. Freelancer delivers KiCad project. |
| D5 | Antenna | IPEX to panel-mount SMA pigtail for LoRa. |
| D6 | Interop | Meshtastic protocol on 433 MHz band. |
| D7 | Power Button | Soft-latch circuit via TPS63802 EN pin. Phone-style: hold 3s ON, click = screen toggle, hold 3s OFF. |
| D8 | OLED Mount | **Option A**: 0.96" SSD1306 OLED module direct-soldered to 4-pin plated through-holes on PCB. |
| D9 | Buttons | **Both**: 5× small SMD tactile test buttons on-board + 5× 2-pin plated breakout pads for hand-wired IP-rated external buttons. |
| D10 | USB-C | **Off-board breakout**: USB-C connector mounted to enclosure with 4-wire pigtail soldered to 4-pin PCB breakout pads (VBUS, D+, D−, GND). CC1/CC2 pull-downs remain on PCB. |
| D11 | GPS Antenna | Passive 25mm ceramic patch antenna internally mounted, connected via U.FL. |
| D12 | Power MOSFET | UMW FS8205A replacement part **LCSC C351449** confirmed. |
